# 📚 Exercícios de Algoritmos - Manzano

Este repositório reúne os enunciados, orientações estruturadas e exemplos em pseudocódigo dos exercícios propostos no livro **"Algoritmos: Algoritmos, Estruturas de Dados e Objetos"** do renomado autor **José Augusto N. G. Manzano**. O livro é uma das referências mais consolidadas para o ensino de lógica de programação e algoritmos no Brasil.

---

## 🎯 Objetivo do Repositório

O principal objetivo deste projeto é servir como um **guia didático e estruturado** para quem está iniciando ou aprimorando seus conhecimentos em lógica de programação. Cada exercício fornece:
1. **Descrição detalhada** do problema.
2. **Passo a passo lógico** para a solução.
3. **Exemplos práticos** de Entrada, Processamento e Saída.
4. **Resolução de referência em pseudocódigo**.

Você pode utilizar estes enunciados como base para praticar e implementar as soluções em qualquer linguagem de programação de sua preferência (como Python, C, C++, Java, JavaScript, Portugol/Visualg, etc.).

## 📝 Anatomia de um Exercício

Todos os exercícios seguem uma estrutura padronizada para facilitar a compreensão. Por exemplo, o arquivo [Exercicio01.md](file:///c:/Users/ferreira-mr/Projects/Exercicios_LivroAlgoritmos_JoseAugustoManzano/01_EstruturaSequencial/Exercicio01.md) contém:

```markdown
# 📝 Exercício X: [Título do Exercício]

## 📖 Descrição
[Explicação detalhada sobre o problema a ser resolvido]

## 🚶 Passo a Passo
1. Entrada de Dados: [O que precisa ser lido]
2. Processamento: [Fórmulas e lógicas aplicadas]
3. Saída de Dados: [O que deve ser exibido]

## 🧪 Exemplos
- Entrada: X -> Processamento -> Saída: Y

## 💻 Exemplo em Pseudocódigo
```plaintext
[Resolução conceitual do algoritmo]
```
```

## 🚀 Como Utilizar para Estudar

1. **Escolha um Capítulo**: Comece do capítulo 1 se for iniciante, ou vá direto para o tópico que deseja revisar (ex: Matrizes ou Subprogramas).
2. **Leia o Enunciado**: Abra o arquivo `.txt` do exercício escolhido e analise o passo a passo lógico e os exemplos de entrada/saída.
3. **Escreva seu Código**: Crie uma pasta ou arquivo de código separado em sua máquina e tente implementar o algoritmo.
4. **Compare com o Pseudocódigo**: Veja o pseudocódigo sugerido no final do arquivo de enunciado para comparar com sua própria lógica e validar a resolução.
